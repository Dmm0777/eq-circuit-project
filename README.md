This is a personal project designed, built and tested by me. The objective of the circuit is to help shape the signal of an electric guitar before amplification. 

The design implements 3 band pass filters in parallel with center frequencies of approximately 90, 900 and 9000 Hz for bass, middle and treble respectively.
It employs 3 operational amplifiers to boost or cut each band and is thus an active EQ circuit. 

This design includes an input buffer to ensure the signal sees a high input impedance, preventing loading. It also includes local decoupling capacitors between the power rails
and the op-amp power pins to ensure power stability. 

The project has already completed its design and simulation phases is currently in the prototype testing phase. More updates will be made to documentation soon.
